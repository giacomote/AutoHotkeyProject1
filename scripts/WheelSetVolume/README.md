# WheelSetVolume
A script that lets you control the volume using the mouse wheel.

After launching the script, you can press the `Win` key, and use your mouse wheel to:
- Increase the volume
- Decrease the volume
- Mute / unmute the volume

## Usage notes
The script can be launched in a silent mode (without sending the notification at startup) using `-s` or `--silent`
options.
