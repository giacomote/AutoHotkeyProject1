# Scripts

## MouseRemote
A script that lets you use your mouse as a remote control to navigate between slides (previous and next) while using
slideshow applications (such as PowerPoint).

For more details about the script, its functions and the ability of personalizing it to your preferences, please refer
to its [README](MouseRemote/README.md) file.

## ProcessGetInfo
A script that lets you know some info associated with a specific window (process name and window title).  
It can be useful to configure other script.

For more details about the script and its functions, please refer to its [README](ProcessGetInfo/README.md) file.
